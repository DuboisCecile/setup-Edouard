# Keyboard Shortcut for VS Code (macOS)

- File navigation:
  `cmd + p / cmd + e`

- Side panel navigation:
  `cmd + shift + e`

- Toggle side panel:
  `cmd + b`

- Organize imports:
  `shift + alt + o`

- Comment line:
  `cmd + :` (remapped to `shift + cmd + :`)

- Sort lines:
  `cmd + alt + s`

- Terminal:
  `cmd + Ã¹` (remapped to `cmd + $`)

- Multiple cursors:
  `cmd + alt + up/down arrow`

- Search:
  `cmd + f`

- Replace:
  `cmd + h`

- File search:
  `cmd + shift + f / h`

- Duplicate:
  `shift + alt + up/down`

- Go down one line:
  `alt + up/down`

- Multiple cursors:
  `alt + clic`
  `cmd + alt + up/down`

- Go up / down:
  `cmd + home / end`

- Delete previous word:
  `alt + enter`

- Delete/cut a line:
  `cmd + x`

- Select entire words:
  `cmd + shift + left / right`
